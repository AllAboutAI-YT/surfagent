Local notes
-----------

We do not depend on puppeteer-extra-plugin-stealth as an npm package because it
peer-depends on puppeteer-extra → puppeteer (~60 MB), which is dead weight for a
direct-CDP tool. The evasions in src/stealth/payload.ts are hand-adapted from
the upstream evasions/* tree (navigator.webdriver, chrome.runtime, chrome.app,
chrome.csi, chrome.loadTimes, navigator.plugins, navigator.languages,
navigator.permissions.query, navigator.vendor, navigator.hardwareConcurrency,
WebGL vendor/renderer, iframe.contentWindow, media.codecs, window outer
dimensions). Each evasion in payload.ts carries a `// @evasion: <name>` header
so future Chrome-version drift is debuggable.

Maintenance: when a real block incident points at a stale evasion, refresh
against the upstream tree at the commit pinned above.
